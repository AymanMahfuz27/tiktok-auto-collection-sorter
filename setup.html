Let's get set up

Point this tool at your TikTok videos folder and it'll learn to sort them automatically.

1
Export your TikTok data — skip if you already have the files

In TikTok: Profile → ☰ → Settings → Privacy → Personalization and data → Download your data.
Choose Video format, then extract the downloaded zip.

What you'll get: A folder called something like TikTok Data containing Favorites/videos/ with your saved .mp4 files.
2
Point to your videos folder

Paste the full path to the folder that contains your .mp4 files. Subfolders become your sort categories.

💡 Windows: open the folder in Explorer, click the address bar, copy the path.
    Mac: right-click the folder → Get Info → copy the "Where" path, or drag the folder here.
